# Module 1: Introduction to Medical Imaging on AWS

## Overview

Medical imaging AI is transforming healthcare, enabling faster and more accurate diagnoses. This module introduces the challenges of training medical imaging models and how AWS services address them.

## Learning Objectives

- Understand medical imaging AI challenges
- Learn AWS services for ML workloads
- Set up your workshop environment
- Explore the datasets

## Duration

30 minutes

## Topics

1. Medical Imaging AI Landscape
2. AWS ML Services Overview
3. Environment Setup
4. Dataset Exploration
